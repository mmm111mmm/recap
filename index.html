Week 4 recap material

What is this?

These are the topics you need for the week 6 project.

The recap consists of javascript source files. In these files you find comments explaining the concepts.

How to use this:

The javascript source files live here https://github.com/newfivefour/recap

You can clone the github repo and play with the files in your editor. Or read them there.

If you clone them, remember to do 'git pull origin master' to get any new changes.

The source files are meant to be read from top to bottom. In the order below.

But I'm including the links below. You can skip to certain parts in the source files.

A note on the comments:

There are a huge number of comments.

If it makes the code clearer to read, feel free to delete them after you understand them.

Important notice:

It's not enough to read and understand. You must practice and play.

A lot of the sytax is new. Feel free to use a reference to remember the syntax.
